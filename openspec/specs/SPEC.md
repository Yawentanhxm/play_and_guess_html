# 你弹我猜 - 音乐节奏猜谜游戏规格说明书

## 1. 项目概述

- **项目名称**: 你弹我猜
- **项目类型**: 网页音乐节奏猜谜游戏
- **核心功能**: 展示简谱并通过键盘指示引导玩家弹奏，弹奏完成后从三个选项中选择正确歌名
- **目标用户**: 音乐爱好者、休闲游戏玩家

- **错误答题**: 红色抖动动画
- **简谱音符**: 当前音符高亮放大 (scale 1.2)
- **背景**: 缓慢移动的星空粒子效果

### 2.4 组件规格

#### 简谱显示区

- 显示当前歌曲的简谱数字
- 当前演奏音符高亮显示（紫色发光）
- 已演奏完成的音符显示为淡色

#### 键盘指示器

- 显示 5 个按键: A, S, D, F, G
- 对应简谱: 1, 2, 3, 4, 5
- 当前需要按下的按键闪烁高亮
- 按下后显示绿色 ✓ 反馈

#### 答题选项

- 3 个矩形按钮，横向排列
- 默认状态: 半透明背景 + 边框
- 悬停状态: 背景变亮 + 轻微上移
- 选中状态: 根据对错显示绿色/红色

#### 计时器

- 圆形进度条显示剩余答题时间
- 倒计时 15 秒
- 时间少于 5 秒时变红闪烁

## 3. 功能规格

### 3.1 核心功能

#### 游戏流程

1. **开始界面**
   - 显示游戏标题 "你弹我猜"
   - "开始游戏" 按钮
   - 简单操作说明

2. **演奏阶段**
   - 播放简谱对应的旋律（使用 Web Audio API 或预录音频）
   - 简谱数字依次高亮
   - 对应键盘按键闪烁指示
   - 玩家按正确按键会播放对应音符
   - 演奏完成后自动进入答题阶段

3. **答题阶段**
   - 显示 3 个歌名选项（1 个正确，2 个干扰项）
   - 15 秒倒计时
   - 玩家点击选项后立即显示结果
   - 正确: 得分 +10，错误: 得分 +0

4. **结果展示**
   - 显示本轮答题结果（正确/错误）
   - 1.5 秒后自动进入下一题

5. **游戏结束**
   - 完成 10 轮后显示最终得分
   - "再来一局" 按钮

### 3.2 数据处理

#### 歌曲数据（示例）

```javascript
{
  id: 1,
  name: "小星星",
  notes: [1, 1, 5, 5, 6, 6, 5],
  tempo: 120,  // BPM
  options: ["小星星", "两只老虎", "生日歌"]
}
```

#### 游戏状态

```javascript
{
  currentLevel: 1,
  score: 0,
  isPlaying: false,
  isAnswering: false,
  currentNotes: [],
  currentNoteIndex: 0,
  timeRemaining: 15
}
```

### 3.3 交互设计

| 交互 | 响应 |
|------|------|
| 点击开始按钮 | 进入第一关，播放第一首歌 |
| 按下正确按键 | 播放音符音效，简谱前进到下一个音符 |
| 按下错误按键 | 播放错误音效，但不前进 |
| 点击答题选项 | 立即判定对错，显示结果 |
| 超时未答题 | 视为错误，自动进入下一题 |

### 3.4 边缘情况

- 玩家在演奏阶段未按任何键: 等待直到演奏结束
- 玩家在答题阶段未选择: 倒计时结束后自动判定为错误
- 浏览器不支持 Web Audio: 显示提示，使用 fallback 方案
